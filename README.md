This HTML file shows all CSS types together in one demo.

- **Inline CSS:** style directly in element using `style=""`. Quick but not reusable.  
- **Internal CSS:** inside `<style>` tag, reusable in same file.  
- **External CSS:** normally separate `.css` file, reusable across pages. Here simulated with a class.  
- **Tailwind CSS:** utility classes in `class=""`, modern and responsive.

Each box has text centered to show differences.  
Key takeaway: Inline > Internal/External for priority, Tailwind is fastest & modern.
